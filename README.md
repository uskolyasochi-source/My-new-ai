> Умный консольный/графический чат-бот с системой лицензирования, обучением и защитой доступа.

## ✨ Возможности

- 🔐 **Система авторизации**: доступ по ключу, генерируемому администратором
- 🧠 **Умный поиск**: нечеткое сопоставление (fuzzy matching) через `difflib`
- 📚 **Обучение**: добавляй новые фразы и ответы в реальном времени
- 💬 **Контекст**: бот помнит последние сообщения для связного диалога
- 🎨 **GUI**: графический интерфейс на `tkinter` (встроен в Python)
- 🔴 **Red Alert Mode**: секретный режим "паники" по команде `kill.os`
- 📊 **Анализ текста**: тональность, статистика, время
- 🗂️ **Модульная архитектура**: легко расширять и поддерживать
